A fibonacci calculator in swift with unit and ui tests

### Prerequisites

* [Xcode](https://developer.apple.com/xcode/) 11.3

### Installing

```
git clone https://github.com/HelioMesquita/FibonacciCalculator.git
open fibonacci.xcodeproj
```
